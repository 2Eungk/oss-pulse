# Example Reports

These examples show `oss-pulse` against public repositories, then a local what-if scan after adding the recommended maintainer surfaces in a temporary clone.

The after reports are not claims about upstream repository state. They show what the score would look like if the listed maintenance files and recent Git author identity activity were added.

Generated on 2026-07-08.
